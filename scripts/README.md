# Scripts

## Deprecation Notice: `backfill-hash-api-keys.ts`

The `backfill-hash-api-keys.ts` script is no longer functional. It was intended to run during a two-phase migration to hash plaintext API keys. However, because the database schema was synchronized using `prisma db push`, the `key` column was dropped immediately, making the historical plaintext keys unrecoverable. 

The migration history has been reconciled via `20260722114600_reconcile_api_key_hash_schema`. Any API keys created prior to this drop must be manually regenerated by the merchants.
